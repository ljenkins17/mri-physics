## Projects

### [`mri_reconstruction/`](./mri_reconstruction/)
Simulation of 2D MRI k-space data acquisition and FFT-based image reconstruction. Covers SNR measurement and the effects of k-space undersampling on image quality.

```bash
cd mri_reconstruction
pip install -r requirements.txt
python3 kspace_reconstruction.py
```
